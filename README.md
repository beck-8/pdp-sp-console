# PDP SP Console — design proposal

PRD + interactive wireframe + issue breakdown for an operations console for PDP storage providers, built into Curio (a breakdown of [filecoin-project/curio#1320](https://github.com/filecoin-project/curio/issues/1320)).

- **[PRD.md](PRD.md)** — problem, model, core decisions, data mapping (appendices)
- **[TASKS.md](TASKS.md)** — EPIC issue breakdown (M0–M6)
- **[mock/](mock/)** — interactive UX mock, single static HTML file, **fake data only** — [view live](https://beck-8.github.io/pdp-sp-console/mock/)

All numbers, addresses, and names in the mock are invented for illustration.
